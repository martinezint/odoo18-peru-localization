# Odoo 18 CE — Localización Perú

Localización **completa, open source y mantenida** para Odoo 18 Community Edition.

## ¿Qué cubre?

- **Contabilidad**: PCGE 2019 completo, impuestos peruanos (IGV, ISC, ICBPER, IVAP), retenciones, percepciones, detracciones.
- **Facturación electrónica SUNAT**: factura, boleta, NC, ND, RC, RA — UBL 2.1 + XAdES-BES.
- **Guía de Remisión Electrónica GRE 2.0**: REST + JWT, remitente y transportista.
- **Libros**: SIRE (prioritario) + PLE (compatibilidad).
- **POS**: extensión del POS core con boletas EDI offline-safe.
- **Maestros**: validación RUC, consulta SUNAT/RENIEC online, catálogos completos.

## ¿Por qué?

Odoo Enterprise tiene `l10n_pe_edi` pero **es propietario y de pago**.
OCA/l10n-peru lleva años estancada (ramas 17.0 y 18.0 vacías).
Este proyecto cubre ese gap con licencia **AGPL-3**.
